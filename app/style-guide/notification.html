Notifications

A notifications api exists, the module is named notification and provides the following functions.

  • warn(message, options) - which broadcast notifications:warn
  • info(message, options) - which broadcast notifications:info
  • success(message, options) - which broadcast notifications:success
  • error(message, options) - which broadcast notifications:error
  • dismissAll() - which broadcast notifications:dismissAll

Options

All of the options have defaults, and do not need to be passed in


{
    timeout: 10, //a value in seconds - optional default 10
    scope: $rootScope, //a scope to use in the notification, message will be on the scope
    templateUrl: 'test.html', //an optional template to render
    resolve: {
        test: ['$q', function($q) {
            var defer = $q.defer;
            return defer.promise;
        }]
    }, //members that will be resolved and passed to the controller as locals
    controller: 'testCtrl' //optional controller (a string or controller function)
}
