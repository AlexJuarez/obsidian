Select2

A custom select2 directive exists! select2 and allows for the following attributes

warning: do not wrap select2 elements in label, on mobile it will cause the select to open

  • multiple - makes the select a multiselect
  • select2 - options hash
  • ng-modeloptional - binds to a model
  • s2-options | ng-options optional - binds to a value collection see ngOptions for documentation.
  • ng-model-optionsoptional - example ng-model-options="{ getterSetter: true }"

Options

all of the options can be defined either as attributes on the element or in the select2 options hash


{
    placeholder: 'a string for the when empty state',
    limit: 25, //a render limit for the dropdown,
    searchLimit: 5, //the number of search results to return
    minimumResultsForSearch: 10, //The thresh hold before displaying a search box
    tokenSeparators: [','], //an array of separators for multiselect
    tags: false, //enabled to allow custom tag creation. the created tags are added to the values array
    query: function(data, term, callback) {}, //returns the drop down results
    width: {*}, //see width info for details
    allowClear: false, //allow the client to remove their selection
    formatModelInsert: function(v) { return { value: v.id, name: v.text }; } //returns the correct formatting for values insert on new tag creation.
}

Multi-Select

tags and query can not currently enabled at the same time.

{{multiselect}}


<select select2 multiple tags="true" ng-model="selected" s2-options="item.name for item in multiselect track by item.value">
</select>
