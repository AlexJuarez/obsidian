Responsive Table Example

In order for the responsive table collapsing to work a little bit of set up is required. TD's need a data-header attribute, and .table-xs-scroll class. 


<div class="table-wrapper table-xs-scroll">
    <table class="table">
        <thead>...</thead>
        <tbody>
            <tr>
                <td data-header="...">...</td>
            </tr>
            ...
        </tbody>
    </table>
</div>
